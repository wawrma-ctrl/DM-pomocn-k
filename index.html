🔗 Připojení Google Drive
1
Otevři console.cloud.google.com → vytvoř nový projekt
2
APIs & Services → Library → vyhledej Google Drive API → Enable
3
Credentials → Create Credentials → OAuth 2.0 Client ID → typ: Web application → do Authorized JavaScript origins přidej URL aplikace (např. https://tvoje.github.io)
4
Zkopíruj Client ID a vlož sem:
Client ID se uloží jen do localStorage tohoto prohlížeče.
